Connect to Snowflake Postgres

Connect to your Postgres instance using psql or a graphical client.

Connect with psql

Use the connection string from the instance creation step:

Shell 
# Replace the placeholder values with your actual instance details
psql "postgresql://snowflake_admin:YOUR_PASSWORD@YOUR_INSTANCE_HOST.YOUR-ACCOUNT.YOUR-REGION.aws.postgres.snowflake.app:5432/postgres?sslmode=require"

Connect with a GUI (DBeaver/pgAdmin)

ParameterValue

✅ Expected Result

You should see a psql prompt like:

psql (17.x)
SSL connection (protocol: TLSv1.3, cipher: TLS_AES_256_GCM_SHA384)
Type "help" for help.

postgres=>
