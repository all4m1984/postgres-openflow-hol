Deploy the PostgreSQL CDC Connector

Deploy the Openflow PostgreSQL connector into your runtime.

The Openflow runtime from Step 6 is now active. Let's install the PostgreSQL CDC connector from the Openflow home page.

Install the PostgreSQL Connector

  1. Navigate to the Openflow overview page (the Openflow home page, not the canvas)
  2. In the Featured connectors section, click View more connectors
  3. On the Openflow connectors page, find PostgreSQL and click Add to runtime
  4. In the Select runtime dialog, select openflow_lab_runtime from the Available runtimes dropdown and click Add
  5. Authenticate with your Snowflake credentials when prompted, then click Allow to authorize the runtime application to access your Snowflake account
  6. Authenticate to the runtime with your Snowflake credentials
✅ Expected Result

The Openflow canvas opens with the PostgreSQL connector process group added to it. The connector installation takes 1-3 minutes to complete.
