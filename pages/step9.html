Start the CDC Pipeline

Launch the pipeline and monitor the initial data snapshot.

With all parameters configured and verified, start the CDC pipeline.

Start the Flow

  1. In the Openflow connector page, click "Start" (the play button)
  2. All processors should transition to Running state
  3. The initial snapshot will begin capturing existing data
ⓘ
Initial Snapshot

When using Ingestion Type = full, the connector first performs a complete snapshot of existing data, then switches to streaming incremental changes. For our small dataset, this completes in under a minute.

Monitor Pipeline Status

Check the connector status in the Openflow UI:

MetricExpected Value
Running Processors> 0
Invalid Processors0
Bulletin Errors0
✅ Expected Result

The connector shows all processors in Running state with no errors. The data flow graph shows records moving through the pipeline.
